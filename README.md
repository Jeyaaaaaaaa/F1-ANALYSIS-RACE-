# F1-ANALYSIS-RACE-
A project that analyses two drivers' performaces in a race which is chosen by the user.
Programming language:Python
OS:Windows 11
Libraries used:Seaborn,Pandas,FastF1

OVERALL DESCRIPTION:
-A GUI that allows a user to enter a year and race
-They can choose two positions in the race to compare
-They then can view graphs on average sector times and lap times, as well as a comparison of speeds on their fastest lap

TECHNIQUES UTILISED:
-OOP
-THREADING

KEY TAKEAWAYS:
-The driver in 1st place isn't usually the fastest.
-Usually slow laps are due to safety cars
-Tracks have significanyly longer sector times than other tracks
